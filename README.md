## О нас

Приветствуем в **Javas Organization** – сообществе разработчиков, увлеченных Java-технологиями и современными подходами к разработке программного обеспечения.

## 🎯 Наша миссия

- Разработка open-source проектов на Java
- Обмен знаниями и опытом
- Создание качественного кода с использованием лучших практик
- Поддержка начинающих Java-разработчиков

## 🔧 Технологии

- **Языки:** Java 17+, Kotlin
- **Фреймворки:** Spring Boot, Micronaut, Quarkus
- **Базы данных:** PostgreSQL, MySQL, MongoDB
- **Инструменты:** Maven, Gradle, Docker, Kubernetes
- **Тестирование:** JUnit 5, Mockito, Testcontainers
